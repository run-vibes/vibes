# Design Prototypes

Static HTML/CSS prototypes exploring visual directions for vibes.

## Phases

### Phase 01: Exploration

Initial design exploration across multiple visual directions:

- **CRT/Retro**: Phosphor green, scanlines, terminal aesthetic
- **Neo-Brutalist**: Raw, bold typography, stark contrasts
- **Luxury Noir**: Dark, sophisticated, premium feel
- **Sci-Fi Console**: Futuristic control panel aesthetic

28 prototypes covering dashboard layouts, session views, and component variations.

### Phase 02: Art Deco (Current)

Milestone 50 home page redesign using Art Deco visual language:

- **Typography**: Titillium Web (display), Karla (body), Barlow Condensed (labels)
- **Colors**: Navy base, gold accents, teal highlights
- **Elements**: Diamond status dots, gold end-cap progress bars, geometric patterns

Key files:
- `01-full-dashboard.html` - Complete dashboard with all zones
- `07-dashboard-final.html` - Final refined version
- `shared.css` - Design system tokens and components

## Viewing

Open any HTML file directly in a browser, or serve locally:

```bash
cd docs/design/prototypes/phase-02-art-deco
python -m http.server 8765
```

Then visit http://localhost:8765
